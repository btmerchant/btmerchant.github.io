    The Lessons

    HTML CSS and JavaScript

    Learning HTML as well as other languages is difficult mostly because of the mountains of syntax there is to memorize, and in-fact you would need to be one of Stan Lee’s SuperHumans to accomplish that. Fortunately we are not expected to commit it all to memory, just enough to be Dangerous! I figure it is like this.. this code was not written to be confusing, I am sure the authors made every attempt possible to make it as intuitive as they could given the obvious constraints. Therefore the more of the semantics you learn the closer you get to reaching that point where your brain begins to learn how to interpret the code, and even if you haven’t learned a peticular command it will become more and more intuitive as you progress, in other words “Baby Steps”, learn to walk and after a while you will begin to run. However we can’t all be Olympian track stars…
